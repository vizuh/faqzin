Lightweight FAQ shortcode.

== Description ==
Use `[faqzin] ... [/faqzin]` to wrap your FAQ markup. Add `.faqzin-item`, `.faqzin-question`, and `.faqzin-answer` classes inside for toggles, and pass `class="your-custom-class"` to the shortcode to set extra wrapper classes.
